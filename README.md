# Análisis de Tarifas de Telecomunicaciones - TripleTen - Bootcamp Análisis de Datos
El proyecto tuvo el objetivo de identificar cuál de las dos tarifas de prepago (Surf y Ultimate) genera más ingresos para ajustar el presupuesto de publicidad. 

Para ello, procesé bases de datos para identificar patrones de consumo, hice pruebas de hipótesis para comparar los ingresos entre los planes y entre regiones, y elaboré visualizaciones para comparar los ingresos entre los planes y entre regiones.

Con base en los hallazgos identificados, se sugirió que la estrategia de marketing de la compañía debe enfocarse en promover la contratación del plan Surf ya que la gente suele contratarlo más a pesar de que consuman una cantidad de GB superior y paguen un poco más en su renta mensual.

Herramientas utilizadas: Python
