# An-lisis-Tarifas-Telecomunicaciones
El presente proyecto tiene el objetivo de identificar las tendencias de tarifas de una compañía de telecomunicaciones. 
Análisis exploratorio y prueba de hipótesis de dos planes de prepago de una compañía de telefonía | Junio 2025
o	Procesamiento de bases de datos para identificar patrones de consumo. 
o	Elaboración de visualizaciones para compartir los hallazgos
o	Prueba de hipótesis para comparar los ingresos entre los planes y entre regiones. 
o	Herramientas utilizadas: Python
